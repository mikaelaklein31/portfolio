Django 1

Today we were introduced to MVC, frameworks, and Django. Model-view-controller paradigm (MVC) is a way to relate user interface to underlying data models. A framework is different from a library because a library is a premade collection of functions/methode/etc., while a framework is a collection of what is used in a certain application. I'd prefer a framework over creating my own app to keep my work organized/legible to others and efficient.
